Sales Data Analysis Report

Project Overview and Objectives

The objective of this project is to analyze a sales dataset using Python and the Pandas library. The program reads data from a CSV file and generates useful insights such as total revenue, average sales, highest sale value, and the best-selling product. This project helps in understanding how data can be analyzed to support business decisions.

What I Learned

Through this project, I learned how to use the Pandas library to work with real-world datasets. I learned how to load CSV files, explore data, check for missing values, calculate important statistics, and generate reports based on the data. This project improved my understanding of data analysis using Python.

Setup and Installation Instructions

1. Install Python on your computer.
2. Install Pandas using the following command:

pip install pandas

3. Save the following files in the same folder:

   * sales_analysis.py
   * sales_data.csv
   * analysis_report.md
   * requirements.txt

4. Open Command Prompt or Terminal.

5. Navigate to the project folder.

6. Run the program using:

python sales_analysis.py

Code Structure Explanation

sales_analysis.py

Contains the main program logic. It loads the dataset, checks data quality, calculates metrics, and generates a report.

sales_data.csv

Contains sales records including product details, quantities, prices, customer information, region, and total sales.

analysis_report.md

Contains project documentation, analysis steps, and findings.

requirements.txt

Lists the required Python library used in the project.

Technical Details

The project uses the Pandas library to:

* Read data from a CSV file
* Display dataset information
* Check for missing values
* Calculate total revenue
* Calculate average sales
* Find the highest sale value
* Identify the best-selling product

Analysis Steps

1. Load the dataset using Pandas.
2. Display the first few rows of data.
3. Check dataset size and structure.
4. Verify missing values.
5. Calculate important business metrics.
6. Generate a formatted analysis report.

Results and Findings

* Total Revenue: ₹12,365,048
* Average Sales: ₹123,650.48
* Highest Sale: ₹373,932
* Best Selling Product: Laptop

The analysis shows that Laptop is the best-selling product based on the total quantity sold. The dataset contains no missing values and is suitable for analysis.
